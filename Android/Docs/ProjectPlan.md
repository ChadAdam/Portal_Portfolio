- Roles

| Role               | Description                                                  |
| ------------------ | ------------------------------------------------------------ |
| Architect          | Reviews all requirements and design artifacts. Ensures the current system design meets all requirements. Leads team in performing design adjustments in response to changes in requirements and/or testing results. |
| Project Manager    | Creates project plan to describe activities, budget, and schedule to move from current state to desired state as described in project charter. Monitors both schedule and cost budgets during project implementation and takes corrective action when budgets are exceeded. Present Designs, project plan, progress metrics, and deliverables to stakeholders for approval. |
| UI Designer        | Designs presentation layer of application to meet all external requirements while providing maximum UX and application response. |
| Developer          | Interpret design documents and produce executable code. Perform alpha testing to ensure code executes without errors and meets design specifications |
| Compliance Manager | Executes testing of deliverable. Performs direct tests in addition to managing beta testing a on going field tests. Interprets results of tests to determine if deliverable is conforming to design specifications and operating correctly. |
